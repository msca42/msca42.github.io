Bubble Sort

发表于 2019-05-09 | 分类于 Algorithm and Date Structure

Bubble Sort is the sort which need compare with each other from the first number . By this way ,the max will turn to the last location in Array or other container .

I will show the code and explain it.

In the beginning , we have many numbers . while we sort one time,the number have decreased one time . In the last we will have the only one .every sort we will reduce the max .
