Java概念盘点（二）

发表于 2020-10-19 | 更新于 2020-10-23
本文字数： 2.7k | 阅读时长 ≈ 2 分钟

数据类型

  • Java的数据类型分为两种，一个是基本数据类型(值类型，内置数据类型)，一个是引用数据类型。

基本数据类型

  • Java在创建之初就引入了基本数据类型，目的是用来支持数值计算，因为基本数据类型可以在执行效率和内存使用两方面提升软件性能。
  • 基本数据类型有四类八种，分别是整型，浮点型，布尔型和字符型。整型又分为byte,short，int,long四种，浮点型又分为float,double两种，布尔型为boolean,字符型为char。
  • 当我们谈论基本数据类型的时候，我们会从这几个方面去说:应用场景，取值范围(值域)，创建时间，创建空间，类型转换。下面我们一个一个来说。

  • 首先对于数据类型，它的应用场景在于修饰变量。那么创建时间与创建空间也就是变量创建的时间与空间。变量在Java中可以分为这样几类:类变量，实例变量，局部变量。

    • 类变量:独立于方法之外的变量，用static修饰，存放在方法区中，属于共享变量

    • 实例变量:独立于方法之外的变量，没有static修饰，存放在堆中，通过GC回收，属于对象的成员变量，根据访问控制符的不同进行设置

    • 局部变量:类的方法中的变量，声明在方法，构造方法或者语句块中，没有默认值，需要手动初始化,在栈中生成消失

  • Java基本数据类型存储在什么地方？

    • Java的基本数据类型都在栈中，存在于堆重点为基本数据类型自动装箱的包装类

      从byte到整型的取值范围

  • 依据赋值给不同类型的变量，基本数据类型的应用场景也限于类，实例和局部这三部分。众所周知，物理层传输的是bit(比特)流，8bit相当于1byte(字节),硬盘中的存储单位为byte。在Java中数据的运算是通过补码进行计算的。
  • 具体说明：
    • 首先2的8次方为256，即byte的取值范围为正负256之间，由于0的存在，所以在-128—-127或者-127—-128；我们知道Java中所有正数都是有符号位的，8位符号位最大值为0111 1111 为127，1111 1111为-127；现在假设byte占用9位而不是8位，那么如何表示+(-)128呢？ 0000 1000 0000 为128； 0001 1000 0000为-128；正数的补码为其本身，负数的补码运算先将原码符号位不变，其余取反，然后在其(反码)基础之上+1；即0001 0111 1111 + 1 也就是0001 1000 0000 与其原码相同。所以计算机就取1000 0000 -0为-128；
