点石成金读书笔记(1)

发表于 2020-04-25 | 分类于 睡前阅读
本文字数： 2.8k | 阅读时长 ≈ 3 分钟

前言与导读

  • 从今天开始，我将会更新这个睡前阅读系列的读书笔记，通过这样的方式养成阅读习惯。

  • 这是本系列的第一篇，是一本叫做点石成金的书，从昨天的导读进度来简要说下这本书主要内容。本书的作者是一名UX(用户体验)设计师，本书也是这方面的内容介绍。昨晚，我阅读的范围是截至到第一章第五页。

  • 前言部分主要讲了由于移动互联网的兴起，本书之前的两个版本也有些过时，所以作者做出了一些更新。但它仍是一本让网站设计的卓越有用的书。[1]

反智与不变

  • 黑格尔的《历史哲学》中说到: 经验和历史所昭示我们的，却是各民族和各政府没有从历史方面学到什么，也没有依据历史上演绎出来的法则行事。[2]放到这里，从作者关于可用性与用户的使用习惯方面看，从古至今，这种变化都是微乎其微的。即此时我所理解的UX可用性是尽量还原用户本身的习惯，本书还提到了youtube上的一个挪威人看书视频，联想过去自己的举动 ，总有种荒诞可笑的意味。就目前的我来说，这种反智主要体现在人自己不去尝试，不去鼓捣，永远觉得必须带着攻略玩游戏，看着别人做完一遍之后自己在动手。边界用于现定于别人的讲述，而不是事物本身的特点。不会翻书，不会正面翻，不会反面翻，不会往回翻。

一些建议

  • 此外，作者为了方便本书的读者，也说明了两个原因[3]

    • 如果它很薄，就更有可能得到
    • 你不需要面面俱到

  • 首先，对于第一条的解释，作者是这样论述的:某个东西越需要投入大量时间，或者看起来会这样，将来就会越少使用。我是这样理解，越是庞大的东西或计划，如果不能拆分成举手这样让人一下就办到的细细分组，就很有可能半途而废，就像类似手机app一样，可以刷两三个小时抖音，可以看两三个小时咨询，视频，此时大脑就处于一种放松的真空状态，什么都不想，感觉很舒服。但是看书看十分钟或一个番茄钟，就感觉时间很漫长。读书也是一个道理，一本书很短，快一点半天就读完和玩一个月读完一本书的感觉是完全不一样的，我很久之前在新华书店，半天可以读完一本东野圭吾的小说，感觉是酣畅淋漓，好不痛快。但是学英语十多年，还是个残废。

  • 其次对于第二条的理解，简单来说就是T型人才，关于通才和专才的争论由来已久，但是并不是每个开车的司机就需要懂得其内部构造，运行原理，因为代步才是主要需求。对于开发来说，上游产品，测试，下游运维，营销了解基本就行。

预言

  • 作者还提到了关于预言的话，我觉得有必要拿出来说说。作者是这样描述的：预言中真正重要的哪些往往会让我们觉得意外，哪怕它们在时候看起来相当明显。[4]预言是什么？简单来说就是对已知信息的一种整合出来的结论对未来趋势的一种预判。显然，错误的预言来源于我们狭窄的信息源和固有的常识。预言与恐怖类似，都是超出我们认知范围的事情，能够做出正确预言的人，要么是通过有价值的信息源(如大数据或现实存在的蛛丝马迹)来进行推测的，就好比马亲王波诡云谲的三国。

可用性

  • 让一个平均能力和经验的人(甚至稍低于平均水平)能明白如何使用它，不必付出过度的努力，或者遇到不必要的麻烦。[5]

前言和导读做了一件什么事

  • 借用作者的可用性来说，读者的范围不受限制，阅读本书不是一件难事，和喝水吃饭呼吸一样简单。本书主要讲了介绍网站设计的事，也升华到了一般设计的原理。
