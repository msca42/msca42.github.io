​ background： Now I’m reading a book called 《Algorithm in cartoon》,while editor talk about Space Complexity , a problem hit my head. So I want to summarize it.

​ The problem is written that there are many numbers . Among them ,one is same as the other. We must pick it out.

​ First , I want a container to put these number . Array is useful in picking and selecting. So let’s put them in a array for the time being. Usually , We have two choices in picking the same. One is circulating two arrays. this choice is same as circulate one array twice. Its space complexity is n^2^ .The other is circulate one array with a temp.

​ For low space complexity ,we must choose behind one. So the problem change to the temp is what.we may fill something in the blanks to help us picking . Now , I still don’t know the same one .if I fill all numbers in it , that will turn to be the first choice. Therefore, the temp must be the number is not the number the problem provides and must have a connection with these number . Last of the last ,Key-Value hit me.

​ This problem has a necessary point . The appearance of the numbers , therefore ,the temp or the value may be it.
